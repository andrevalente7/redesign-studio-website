# Brand

## Direction

- Raycast-inspired dark product interface
- Near-black canvas with a surface ladder from `#07080a` to `#121212`
- Inter / system sans typography with `ss03` font features enabled where supported
- Compact command-palette chrome, tight 6-16px radii, and hairline borders
- Orange primary CTA, transparent secondary actions, and glassy orange accents
- No red stripe motif at the top of the page
- Proof-led header with trusted-by logos and three compact credibility stats

## Typography Notes

The site uses a browser-safe stack that approximates the Raycast design reference:

- Display and body: Inter / Helvetica Neue / Helvetica / system sans
- Font features: `calt`, `kern`, `liga`, and `ss03`
- Utility labels: the same sans stack for a product UI feel
